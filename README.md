# MyQuadcopter

Board : STM32F4

operating system : FreeRTOS

hardware sensors : ESP8266 , MPU9250 , TF02

control algorithm : double loop pid controller

x , y 軸平衡使用角度角速度雙環 , z 軸定高使用高度速度雙環
